차이점 분석

AI Default 문제점

  • 1. purple→indigo→blue 그라디언트 히어로
  • 2. 장식용 blob (blur-3xl 원형 2개)
  • 3. 그라디언트 텍스트 (bg-clip-text)
  • 4. glassmorphism 버튼 (backdrop-blur + white/10)
  • 5. gradient CTA (purple→pink) + shadow
  • 6. hover:scale-105 버튼
  • 7. bento grid (col-span-2 비대칭)
  • 8. 카드마다 다른 그라디언트 색상 (4가지)
  • 9. rounded-3xl 과도한 radius
  • 10. 중앙 정렬 히어로 (generic)

Reference-based 해결

  • 1. 솔리드 화이트 배경 (KR-2: #fafafa)
  • 2. 장식 요소 0개
  • 3. 순수 텍스트 (타이포그래피가 디자인)
  • 4. 솔리드 인디고 pill CTA (KR-2 스타일)
  • 5. 단색 버튼 + border 버튼 (GL-3 스타일)
  • 6. hover:bg-color 만 (transition-colors)
  • 7. 균일한 3x2 그리드 (같은 크기)
  • 8. 전체 카드 동일 스타일 (white + border)
  • 9. rounded-md (6px) + rounded-full (pill CTA)
  • 10. 좌측 정렬 히어로 (GL-3 자신감)
