에이전트 시스템

각 에이전트가 전문 역할을 수행하며, Supervisor가 전체 흐름을 조율합니다.

Ideation Agent

요구사항을 분석하고 구현 전략을 제안합니다. 컨텍스트를 이해하고 최적의 접근법을 도출합니다.

Decomposer Agent

복잡한 태스크를 실행 가능한 서브태스크로 분해합니다. 의존성과 병렬 처리를 자동으로 판단합니다.

Critique Agent

코드 품질을 검증하고 개선점을 제안합니다. 보안, 성능, 가독성을 종합적으로 평가합니다.

Router Agent

태스크를 적합한 에이전트에 분배합니다. 에이전트 역량과 현재 부하를 고려해 최적 라우팅을 수행합니다.

QA Agent

자동으로 테스트 케이스를 생성하고 실행합니다. 엣지 케이스 탐지와 회귀 테스트를 담당합니다.

Supervisor Agent

전체 워크플로우를 감독하고 에이전트 간 협업을 조율합니다. 오류 복구와 재시도를 관리합니다.
